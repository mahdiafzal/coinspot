## Install

1. Get the sourcecode:
    ```
    $ git clone https://github.com/dashtinejad/aftabmarket.git
    ```

2. Switch to its directory:
    ```
    $ cd aftabmarket
    ```

3. Run `npm install` command, it will install dependencies:
    ```
    $ npm install
    ```

4. Run `npm start` command, it will run project:
    ```
    $ npm start
    ```
